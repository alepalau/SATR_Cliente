SATR - Regional Transportation Management System. SATR is a cross-platform application (originally designed for Android) developed in Python using the Flet framework. It allows drivers to manage their work schedules, record trips and expenses, calculate commissions, and generate detailed PDF reports.

✨ Features

Work Schedule Management: Driver registration, license plate number, and mileage tracking (start/end).

Trip Tracking: Trip recording with payment type (Cash/Bank Transfer).

Expense Management: Expense categorization (Agency, Driver, Miscellaneous).

Automatic Calculations: Commission calculation based on configurable percentages and earnings per kilometer.

PDF Reports: Generation of detailed summaries ready to share.

Data Persistence: Automatic saving of the current work schedule to prevent data loss.

Synchronization: Integration with external servers to display sponsor information.

SATR - Sistema de Administración de Transporte RegionalSATR es una aplicación multiplataforma (diseñada originalmente para Android) desarrollada en Python utilizando el framework Flet. Permite a los conductores gestionar sus jornadas laborales, registrar viajes, gastos, calcular comisiones y generar reportes detallados en PDF.

✨ Características

Gestión de Jornada: Registro de chofer, patente y control de kilometraje (inicial/final).

Control de Viajes: Registro de viajes con tipo de pago (Efectivo/Transferencia).

Gestión de Gastos: Clasificación de gastos (Agencia, Chofer, Varios).

Cálculos Automáticos: Cálculo de comisiones basado en porcentajes configurables y rendimiento por kilómetro.

Reportes PDF: Generación de resúmenes detallados listos para compartir.

Persistencia de Datos: Guardado automático de la jornada actual para evitar pérdida de datos.

Sincronización: Integración con servidores externos para mostrar información de auspiciantes.
